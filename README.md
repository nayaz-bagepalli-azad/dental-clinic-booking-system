# 🦷 Dental Clinic Appointment Booking System

A modern full-stack dental clinic management system built with **Next.js, Supabase, and Tailwind CSS**.

---

## 🚀 Live Features

### 👨‍⚕️ Patient Features
- View doctors list
- Book appointments by selecting:
  - Doctor
  - Date
  - Available time slots
- Prevent double booking
- Instant booking confirmation

### 🧑‍💼 Admin Features
- Secure admin login (Supabase Auth)
- View all appointments
- Approve / Cancel appointments
- Dashboard with appointment stats
- Manage clinic scheduling system

---

## 🧰 Tech Stack

- **Frontend:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Auth)
- **Hosting:** Vercel (ready)
- **Database:** PostgreSQL (Supabase)

---

## 🗄️ Database Schema

- doctors
- appointments
- working_hours

---

## ⚙️ Features Implemented

- Doctor listing from database
- Dynamic slot generation
- Booking system with conflict prevention
- Admin dashboard
- Appointment status management
- Row Level Security (RLS) enabled
- Git version control
